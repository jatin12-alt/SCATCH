## 🚀 Getting Started
### Prerequisites
* Node.js (v16 or higher)
* npm (v8 or higher) or yarn (v1 or higher)
* Vite (v3 or higher)
* Supabase (v1 or higher)

### Installation steps
1. Clone the repository: `git clone https://github.com/jatin12-alt/SCATCH.git`
2. Install dependencies: `npm install` or `yarn install`
3. Start the development server: `npm run dev` or `yarn dev`
4. Open the app in your browser: `http://localhost:3000`

## 🔑 Environment Variables
| Variable | Description | Required |
| --- | --- | --- |
| `VITE_SUPABASE_URL` | Supabase URL | Yes |
| `VITE_SUPABASE_ANON_KEY` | Supabase anonymous key | Yes |

## 🌐 Deployment
1. Create a Vercel account: https://vercel.com/
2. Install the Vercel CLI: `npm install -g vercel`
3. Link your Vercel account: `vercel login`
4. Deploy the app: `vercel build && vercel deploy`
